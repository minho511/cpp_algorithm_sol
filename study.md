    - 2609 최대공약수와 최소공배수 (B1)
        - 유클리드 호제법 (C++)
        ```cpp
        # a, b를 입력으로 받아 최대 공약수를 출력.
        int gcd(int a, int b)
        {
            int c;
            while (b != 0)
            {
                c = a % b;
                a = b;
                b = c;
            }
            return a;
        }
        # 최소 공배수는 두 수의 곱을 최대 공약수로 나눈 값.
        ```
